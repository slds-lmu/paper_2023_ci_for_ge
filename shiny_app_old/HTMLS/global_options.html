Important: The selections that are made here apply to all plots on the website.

It is possible to select a subsets for the data-generating processes (DGPs) and for the inference methods.
By default, we only show a subset for both in the interest of readability:

  • DGP: By default, we don't include the results for chen_10_null (no relatonship between the features and target), as all methods perform poor for this dataset. See the Chen 10 Null tab for a separate analysis.
  • Inference Methods: By default, we omit the
    • out-of-bag and 632+ bootstrap methods for 10, 50, and 100 repetitions, as the CIs are generally far too wide, and the results for 500 repetitions are omitted as they are relatively similar to those with 1000 repetitions,
    • corrected-t method with 50 repetitions (we include the results for 10, 100 repetitions),
    • bayle method with the within-fold variance estimator, as the results are very similar to the all-pairs variance estimator,
    • location shifted bootstrap with 50 repetitions, but we do include the results with 100 repetitions.
