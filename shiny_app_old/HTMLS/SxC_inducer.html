• The plots here visualize the results for a single inducer, aggregated over everything except for the size and what is selected as "Color" or "Group".
  • If there is only one size, the selected "Color" variable becomes the y axis and a point-plot instead of a line-plot is shown.
  • To see the results for bccv, set the min size and max size to 100.
  • To switch between Coverage Frequency and RMSE coverage error, change the selection for "Evaluation".
